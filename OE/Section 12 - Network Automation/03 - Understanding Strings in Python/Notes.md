Python strings

In Python, **strings** are ==immutable sequences of Unicode characters==. They are the primary way to store and manipulate textual data, such as words, URLs, or code.

```
>>> print("test")
test
>>> print('test')
test
>>> print('test' + 'anothertest')
testanothertest
>>> print('test' + '\n' + 'anothertest')
test
anothertest
>>>
>>> print ('this' ' ' 'is' ' ' 'our' ' ' '\n' 'test') 
this is our test
```

\n = new line
